# Last Mile Studio UI 北极星

## 产品定位

Last Mile Studio 是面向 AI 生成 HTML、HTML 演示文稿和 SVG 的源码优先可视化编辑器，是一款专业创作工具。它的外壳应优先服务于反复编辑、检查和导出，而不是呈现为消费级落地页。

设计短语：

> 精准创意工作站

核心品质：

- 值得信赖；
- 技术感强但易于接近；
- 信息密集但可读；
- 只有在状态或风险需要时才引起注意；
- 以画布为中心；
- 适合键盘操作；
- 在 HTML、演示文稿、SVG、源码、Build 和片段工作流之间保持一致。

## 核心用户任务

1. 导入或粘贴源码，并立即理解文档状态。
2. 在不损失画布空间的前提下浏览页面、Build、图层和选择对象。
3. 精确修改几何属性和样式。
4. 检查源码并应用代码，同时不破坏上一次有效的画布状态。
5. 复用片段/组件，同时理解其存储和兼容性。
6. 有把握地预览和导出。

## 层级结构

### 全局层

顶部栏只包含文档/全局操作：

- 产品/文档身份；
- 导入；
- 撤销/重做；
- 适用时的演示预览；
- 导出。

### 导航层

窄活动栏可以在现有的主要上下文之间切换，例如 Pages、Layers 和 Fragments。不得暴露没有真实实现对应页面的目的地。

相邻的上下文面板包含当前导航界面，并保持可调整大小/可折叠。

### 画布层

画布负责：

- 选择对象专属操作；
- 页面/Build 播放状态；
- 画布尺寸；
- 缩放/适应；
- 已编辑文档；
- 紧凑状态信息。

永久工具栏内容应按模式分组。没有选择对象时无意义的控件不应主导空选择状态。

### 检查器层

按任务对现有属性分组：

- Design：几何、文字、填充、描边、外观；
- Build：适用时的 Build 分配和顺序；
- Advanced：类名、内联样式、ID，以及现有行为支持的关联片段元数据。

选项卡或折叠面板不得破坏已输入的值或选择状态。

### 源码层

源码抽屉默认保持折叠。展开后的源码操作应清晰，并保留明确的“应用有效代码”边界。

## 视觉语言

### 表面

使用带有细小、明确亮度层级的中性炭灰色表面。画布应具有最强的视觉对比。永久面板使用边框，而不是浮动卡片阴影。

### 字体

- 产品/UI：系统无衬线字体或已批准的本地字体栈；
- 代码/ID/数字：JetBrains Mono 或现有等宽字体栈；
- UI 正文：12–13 px；
- 元数据/辅助文字：至少 11 px；
- 标题：紧凑，不使用营销级字号。

### 间距

基础间距节奏：4 px。

常用值：4、8、12、16、20、24。

使用密度来支持工作，而不是用它来合理化难以阅读的控件。

### 状态

- 强调色：焦点、选择、激活状态、主要操作；
- 成功：已完成的同步/安全状态；
- 警告：可处理的兼容性或校验问题；
- 危险：破坏性操作和严重错误；
- 禁用：降低强调程度，同时保持标签可读。

### 动效

悬停、焦点、展开/折叠和面板状态使用 120–180 ms 的过渡。在 `prefers-reduced-motion` 下禁用非必要动效。

## 验收目标

在 1280×800、1440×900 和 1920×1080 下：

- 不出现文档级水平溢出；
- 全局导入/导出操作保持可见；
- 画布视口保持可用；
- 主要面板控件不被裁切；
- 不存在没有可访问名称的可见交互控件；
- 控件目标至少 24 px 高，首选 28 px；
- 普通交互文字至少 11 px；
- 焦点状态可见；
- 默认、选中元素、deck、SVG、源码展开、面板折叠和片段库状态都有截图；
- 所有仓库行为门禁通过。
