# Changelog

All notable changes intended for a public release are recorded here. Dates refer to repository history; release tags remain a maintainer decision.

## Unreleased

- Added system-language-aware Chinese and English editor modes, with a top-right `中文` / `EN` switch that translates editor controls, dialogs, status, warnings, and accessibility labels without changing imported document content.
- Refreshed the bundled HTML slide sample with a three-page AfterPrompt overview, 15 Build groups, stable editor IDs, and a v1 editing contract; it opens with all Build content shown by default.
- Fixed dragging a selected nested SVG root and made inline text Apply/Cancel reliable when editor drag handling is active.

## 0.4.0 — 2026-07-20

- Added local-first Visual Fragment packaging, component semantics, compatibility checks, and directory-backed reuse workflows.
- Expanded HTML-slide page/build editing, preview, export, layout parity, and regression coverage.
- Added structured local CLI and JSON-command workflows over the shared document model.
